# Online Store Client

[Heroku live demo](https://bee-seller-react-client.herokuapp.com/)

## Features

This client contains:

- Login
- Marketplace
- Catalog management
- Cart

## Technologies

- Javascript
- React
- React router
- Ant Design

## Usage

To run the app set in the `client/.env` file the ENV `REACT_APP_API_URL="http://localhost:8080"`

```bash
npm start
```
